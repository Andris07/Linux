# 📝 Mi az a Linux?

A Linux egy **ingyenes és nyílt forráskódú operációs rendszer**, amely a **Linux kernelre** épül.

A zárt forráskódú operációs rendszerekkel – például a Windows vagy a macOS – ellentétben a Linux forráskódja bárki számára elérhető. Ennek köszönhetően a fejlesztők szabadon tanulmányozhatják, módosíthatják és továbbfejleszthetik, ami az egyik legnagyobb nyílt forráskódú közösséget hozta létre a világon.

Napjainkban a Linux nemcsak személyi számítógépeken található meg, hanem szervereken, okostelefonokon, felhőszolgáltatásokban, szuperszámítógépeken és számtalan beágyazott rendszerben is.

---

# 📜 Rövid története

A Linux kernelt **1991-ben** készítette **Linus Torvalds**, aki akkoriban a Helsinki Egyetem számítástechnika szakos hallgatója volt.

A projekt kezdetben egy személyes fejlesztésként indult, azonban rövid idő alatt világszerte fejlesztők ezrei csatlakoztak hozzá. Az évek során a Linux az egyik legsikeresebb nyílt forráskódú projektté nőtte ki magát.

Napjainkban fejlesztők ezrei dolgoznak folyamatosan a kernel és a hozzá kapcsolódó szoftverek fejlesztésén.

---

# 🧠 Mi az a kernel?

A **kernel** az operációs rendszer legfontosabb része, amely kapcsolatot teremt a hardver és a szoftverek között.

Feladata többek között:

* 💻 A processzor erőforrásainak kezelése
* 🧠 A memória kezelése
* 💾 Háttértárak kezelése
* 🌐 Hálózati kommunikáció
* 🔌 Eszközmeghajtók (driverek) kezelése
* 📂 Fájlrendszerek kezelése

Kernel nélkül egyetlen alkalmazás sem tudna kommunikálni a számítógép hardverével.

Fontos megkülönböztetni, hogy a **Linux valójában a kernel neve**, míg a teljes operációs rendszereket **Linux disztribúcióknak** nevezzük.

---

# 🐧 Mi az a Linux disztribúció?

A Linux disztribúció (vagy röviden **distro**) a Linux kernelre épülő teljes operációs rendszer.

Egy disztribúció általában tartalmazza:

* A Linux kernelt
* Csomagkezelőt
* Rendszereszközöket
* Grafikus környezetet (opcionálisan)
* Alapvető alkalmazásokat
* Dokumentációt

Néhány népszerű disztribúció:

* Fedora
* Debian
* Ubuntu
* Arch Linux
* Linux Mint

Minden disztribúciónak megvan a saját filozófiája, kiadási modellje, csomagkezelője és célközönsége.

---

# 💻 Hardverigény

A Linux egyik legnagyobb előnye a rendkívüli rugalmassága.

Futtatható például:

* Modern asztali számítógépeken
* Laptopokon
* Szervereken
* Raspberry Pi eszközökön
* Mini PC-ken
* Beágyazott rendszereken
* Szuperszámítógépeken

Egyes könnyű (lightweight) disztribúciók akár **1 GB RAM-nál kevesebb memóriával** is képesek megfelelően működni, míg a modernebb grafikus környezetek természetesen erősebb hardvert igényelnek.

Ennek köszönhetően szinte minden számítógéphez létezik megfelelő Linux disztribúció.
