# 📝 Bemutatkozás

Üdvözöllek a Linux repositorymban!

Ez a projekt a Linux tanulási folyamatomat dokumentálja, és egy folyamatosan bővülő személyes tudásbázisként szolgál.

A célom, hogy minden fontos jegyzetet, telepítési útmutatót, hibamegoldást, parancsot és hasznos információt egy jól átlátható helyen gyűjtsek össze. Bár elsősorban saját tanulásra készül, remélem, hogy másoknak is segítséget nyújt majd a Linux világának megismerésében.

A repository idővel egyre több témával fog bővülni, többek között Linux disztribúciókkal, szerverekkel, virtualizációval, hálózatokkal, homelabbel és egyéb nyílt forráskódú technológiákkal.
