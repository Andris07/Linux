# 📝 About

Before creating a bootable USB drive, you'll first need to download Ventoy and prepare a USB flash drive.

The installation process only takes a few minutes and only needs to be completed once for each USB drive.

---

# 💾 USB Requirements

### ✅ Minimum

* USB 2.0 or newer
* At least **8 GB** of storage

### ⭐ Recommended

* USB 3.0 / USB 3.1 / USB 3.2
* **32 GB** or larger
* Good read and write speeds
* A reliable brand (e.g. SanDisk, Kingston, Samsung)

> **Note:** A faster USB drive can significantly reduce boot times and improve the overall experience, especially when working with larger ISO files.

---

# 🌐 Download Ventoy

Always download Ventoy from the **official website** to ensure you're using the latest stable and unmodified release.

➡️ **Official website:** https://www.ventoy.net

After opening the website, navigate to the **Download** section and select the latest release for your operating system.

---

# 📦 Which Version Should I Download?

Choose the package that matches the operating system you're currently using.

| Operating System | Package                      |
| ---------------- | ---------------------------- |
| Windows          | `ventoy-x.x.xx-windows.zip`  |
| Linux            | `ventoy-x.x.xx-linux.tar.gz` |

This handbook uses the **Windows** version for all installation examples.

---

# ⚙️ Preparing Ventoy

1. Download the latest Windows ZIP archive.
2. Extract the archive.
3. Open the extracted folder.
4. Run **Ventoy2Disk.exe**.
5. If prompted by Windows, allow the application to make changes.
6. Continue with the installation guide.

> ⚠️ **Warning:** Installing Ventoy will erase **all existing data** on the selected USB drive. Make sure to back up any important files before continuing.
