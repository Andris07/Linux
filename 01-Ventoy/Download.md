# 🔽 Download

> Downloading and preparing Ventoy for the first time.

---

# 📝 Overview

Before creating a bootable USB drive, you'll first need to download Ventoy and prepare a USB flash drive.

The installation process only takes a few minutes and only needs to be completed once for each USB drive.

---

# 💾 USB Requirements

### ✅ Minimum

* USB 2.0 or newer
* At least **8 GB** capacity

### ⭐ Recommended

* USB 3.0 / USB 3.1 / USB 3.2
* **32 GB or larger**
* Good read/write speeds
* Reliable brand (e.g. SanDisk, Kingston, Samsung)

> **Note:** Faster USB drives significantly reduce boot and loading times, especially when working with larger ISO files.

---

# 🌐 Download Ventoy

Always download Ventoy from the **official website**.

**Official Website**

https://www.ventoy.net

Navigate to the **Download** section and select the latest stable release.

---

# 📦 Which Version Should I Download?

Choose the package that matches the operating system you're currently using.

| Operating System | Download                     |
| ---------------- | ---------------------------- |
| Windows          | `ventoy-x.x.xx-windows.zip`  |
| Linux            | `ventoy-x.x.xx-linux.tar.gz` |

For this guide, the Windows version is used.

---

# ⚙️ Preparing Ventoy

1. Download the latest Windows ZIP archive.
2. Extract the ZIP file.
3. Open the extracted folder.
4. Run **Ventoy2Disk.exe**.
5. If prompted, allow the application to make changes.
6. Continue with the installation in the next guide.

> **Warning:** Installing Ventoy will erase all existing data on the selected USB drive.
