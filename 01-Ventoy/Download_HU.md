# 🔽 Letöltés

> A Ventoy letöltése és előkészítése az első használathoz.

---

# 📖 Áttekintés

Mielőtt bootolható pendrive-ot készítenél, először le kell töltened a Ventoyt, valamint elő kell készítened egy megfelelő USB pendrive-ot.

A telepítés mindössze néhány percet vesz igénybe, és egy adott pendrive-on csak egyszer kell elvégezni.

---

# 💾 Pendrive követelmények

### ✅ Minimum

* USB 2.0 vagy újabb
* Legalább **8 GB** tárhely

### ⭐ Ajánlott

* USB 3.0 / USB 3.1 / USB 3.2
* **32 GB vagy nagyobb**
* Jó olvasási és írási sebesség
* Megbízható gyártó (pl. SanDisk, Kingston, Samsung)

> **Megjegyzés:** A gyorsabb pendrive-ok jelentősen csökkentik a rendszerindítási és betöltési időt, különösen nagyobb ISO fájlok használata esetén.

---

# 🌐 Ventoy letöltése

A Ventoyt mindig a **hivatalos weboldalról** töltsd le.

**Hivatalos weboldal**

https://www.ventoy.net

Nyisd meg a **Download** menüpontot, majd töltsd le a legfrissebb stabil verziót.

---

# 📦 Melyik verziót válasszam?

Mindig azt a csomagot töltsd le, amelyik megfelel a jelenlegi operációs rendszerednek.

| Operációs rendszer | Letöltendő fájl              |
| ------------------ | ---------------------------- |
| Windows            | `ventoy-x.x.xx-windows.zip`  |
| Linux              | `ventoy-x.x.xx-linux.tar.gz` |

Ebben az útmutatóban a Windows verzió használatát mutatom be.

---

# ⚙️ A Ventoy előkészítése

1. Töltsd le a legfrissebb Windows ZIP csomagot.
2. Csomagold ki a ZIP fájlt.
3. Nyisd meg a kicsomagolt mappát.
4. Indítsd el a **Ventoy2Disk.exe** alkalmazást.
5. Ha a Windows engedélyt kér, fogadd el.
6. A telepítést a következő útmutató mutatja be részletesen.

> **Figyelem:** A Ventoy telepítése törli a kiválasztott pendrive teljes tartalmát.
